# 🚀 DevLens

> **Your projects remember the code. DevLens helps you remember everything else.**

DevLens is a desktop application built for developers who constantly jump between projects and lose valuable context along the way.

Every project accumulates knowledge—bugs you've fixed, setup steps you always forget, API endpoints you spent hours understanding, architecture decisions, deployment rituals, and countless "I'll remember this later" moments. Weeks or months later, that knowledge is gone, and you're back searching AI, old chats, browser history, or digging through commits.

DevLens solves that problem.

Instead of replacing your IDE, DevLens becomes your project's intelligence layer. It sits beside your existing tools and grows with every project, giving you a single place to document, understand, and revisit everything that matters.

## ✨ Features

### 🐞 Bug Book

Build your own searchable knowledge base of solved bugs, including the cause, solution, related files, and lessons learned.

### ⚙️ Project Setup

Never forget how a project is configured again. Save installation steps, development commands, build processes, deployment instructions, and personal reminders for every project.

### 🌐 Endpoint Explorer

Document every API endpoint with its purpose, request and response details, authentication requirements, and the parts of your application that depend on it.

### 🧬 Project DNA

Understand your project at a glance with insights into models, endpoints, services, architecture, and overall project structure.

### 💥 Feature Impact *(Planned)*

Explore how different parts of your application are connected and understand what could be affected before making changes.

## 🎯 Why DevLens?

Because future-you deserves better than:

* "How did I fix this bug before?"
* "What's the command to start this project again?"
* "Where is this endpoint used?"
* "Why did I build it this way?"
* "Didn't I already solve this months ago?"

DevLens helps you preserve your engineering knowledge so every project becomes easier to maintain over time.

## 🛠️ Philosophy

DevLens is **not** an IDE.

It is **not** a project manager.

It is **not** another note-taking app.

It is your **developer memory**, designed to help you spend less time remembering and more time building.

## 🌱 Vision

DevLens starts as a personal companion for developers and grows into a powerful toolkit for understanding, documenting, and navigating software projects with confidence.
